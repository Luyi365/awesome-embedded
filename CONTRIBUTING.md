# 贡献指南

请注意，本项目发布时附带了[贡献者行为准则](./CODE_OF_CONDUCT.md)。参与本项目即表示您同意遵守其条款。

## 为该项目添加新内容

有好的内容或想法想分享？走 Issues 就行，流程很简单 👇

✅ 第一步：先看一眼，别撞车  
去 [Issues](https://github.com/Luyi365/Awesome-Embedded/issues) 页面扫一眼，确认你要提的内容项目里还没有，也没人提过。省得重复劳动 😄

✍️ 第二步：点 "New issue"  
确认无重复后，点击那个绿色的 [New issue](https://github.com/Luyi365/Awesome-Embedded/issues/new) 按钮。

📋 第三步：按模板来，别漏项  
模板里的字段挨个填，提交前检查一遍——内容完整、格式对、没有手误。

🚀 第四步：提交  
检查无误，点击 "Create" 提交，搞定！

⏳ 第五步：等我一星期  
我会在一周内审核你的贡献，通过后就会更新到项目里。

⚠️ 一个小提醒  
目前不接受 PR，统一走 [Issues](https://github.com/Luyi365/Awesome-Embedded/issues) 提交就好。
